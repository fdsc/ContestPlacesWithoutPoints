# Конкурс красоты
# У нас есть три параметра: цвет волос, длина волос, красивая одежда
# Соревнуются пять девушек
# Блондинки лучше русоволосых. Русоволосые лучше брюнеток.
# Длинноволосые лучше коротковолосых
# Красиво одетые лучше, чем некрасиво одетые

# Красивость одежды оценивается жюри из трёх мужчин по баллам: чем больше баллов, тем лучше

# Так как блондинки лучше брюнеток, считаем, что блондинки занимают первое место по цвету волос, русоволосые - второе место, брюнетки - третье место.
# Далее, длинноволосые занимают первое место, средневолосые - второе, коротковолосые - 3-е место.
# Далее указываем бал по одежде с отрицательным знаком
# Так как чем больше баллов - тем лучше. А программа действует по обратному принципу - чем меньше баллов, тем лучше.

# Приоритеты при оценках: цвет волос важнее, чем их длина, длина важнее, чем баллы за одежду.
